# 💰 BucksWise

A modern personal finance tracker built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. BucksWise helps users manage expenses, track spending, monitor budgets, and visualize financial insights through an intuitive and responsive interface.

🌐 **Live Demo:** https://buckswise.vercel.app

---

## ✨ Features

### 📊 Dashboard
- Financial overview cards
- Spending analytics
- Monthly spending chart
- AI spending insights
- Budget tracking
- Savings goal monitoring

### 💸 Expense Management
- Add new expenses
- Edit existing expenses
- Delete expenses with confirmation
- Search expenses
- Filter by category
- Sort by latest, highest, and lowest

### 📁 Export
- Export expenses as CSV

### 💾 Data Persistence
- Local Storage support
- Automatic data saving

### 📱 Responsive Design
- Desktop
- Tablet
- Mobile

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/aditi72067/buckswise.git
```

Navigate into the project

```bash
cd buckswise
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
