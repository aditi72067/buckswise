# Introduction

BucksWise is designed for people who want to improve their financial habits without feeling overwhelmed.

These personas represent the primary users the product is built for.

---

# Persona 1 — The Student

## Profile

**Name:** Aanya Sharma  
**Age:** 20  
**Occupation:** College Student

### Goals

- Track monthly allowance
- Save for gadgets and trips
- Avoid overspending
- Learn personal finance

### Pain Points

- Doesn't know where money goes
- Frequently overspends on food and shopping
- Finds budgeting apps too complicated

### BucksWise Helps By

- Simple expense tracking
- Weekly spending summaries
- Savings goals
- Friendly AI guidance

# Common User Needs

All users want to:

- Understand their spending
- Save more money
- Build healthy habits
- Receive useful financial guidance
- Feel confident about their finances

---

# User Motivations

Users open BucksWise to:

- Check today's spending
- Add a transaction
- Review budgets
- Monitor savings goals
- Receive AI insights
- Track financial progress

---

# Design Implications

Based on these personas, BucksWise should always be:

- Easy to learn
- Visually engaging
- Encouraging rather than judgmental
- Fast to use
- Accessible on mobile and desktop
- Focused on clarity over complexity

---

# Success Criteria

A user should be able to:

- Add a transaction in under 30 seconds
- Understand their financial status at a glance
- Identify overspending quickly
- Receive actionable recommendations
- Stay motivated to build better financial habits
