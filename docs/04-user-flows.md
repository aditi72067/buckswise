# UX Principles

Every user journey should:

- Require the fewest possible steps
- Minimize typing
- Provide immediate feedback
- Prevent common mistakes
- Include loading states
- Handle errors gracefully
- Support mobile and desktop
- Keep important actions within one or two taps/clicks

---

# Success Criteria

A successful flow should allow users to:

- Complete key tasks in under one minute
- Understand what happened after every action
- Recover easily from mistakes
- Feel guided rather than overwhelmed
